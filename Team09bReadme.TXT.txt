This is the readme file for the Team9B Library Management System.

Log in:
Users log in with their staff ID. CIO Venkat is assigned S001- password is 1111
Per system convention, users passwords correspond to their StaffID. For example:
s002- 2222
s003- 3333
s004- 4444

Data entry:
Access the forms from the Menu Strip or the Shortcut buttons. The shortcuts provide convenient access to commonly used functions.

Adding new Members:
Librarian will assign a member id to each new member they enrol. They should input the assigned Member ID number into the form, and the data will be saved to the database.

Reports:
The reporting tools show information such as the following:
Breakdown of members by category (the library has Regular and Premium members)
Breakdown of books by type.
